Open Reader

Open Reader is a web app interface for Project Gutenberg

It provides a more user friendly, clean, and enjoyable interface that allows the user to access Project Gutenberg's extensive library of books that have entered the public domain

Open Reader users can add books to their bookshelves, add bookmarks to these books, and experience a paginated and more "book like" experience, compared to reading the books on Project Gutenberg directly
